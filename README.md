## COVID-19 Trade Impact Analysis
Project Overview

This project analyzes the impact of COVID-19 on international trade between 2019 and 2021. Using Python and data analysis techniques, the study explores trends in imports, exports, commodities, transport modes, and countries to understand how global trade patterns changed during the pandemic.

## Objectives
Analyze overall trade trends before and during COVID-19.
Compare import and export performance across years.
Identify commodities most affected by the pandemic.
Examine the impact on different transport modes.
Discover countries with significant trade growth or decline.

## Dataset
The dataset contains international trade records, including:

Key Insights
Global trade experienced significant fluctuations during the COVID-19 period.
Import and export activities were heavily impacted in 2020.
Several commodities showed strong recovery in 2021.
Trade performance varied across countries and transport modes.
Post-pandemic recovery trends became visible in international trade data.

## Results
The analysis highlights the economic impact of COVID-19 on international trade and demonstrates how data analytics can be used to identify patterns, trends, and recovery indicators from large datasets.
